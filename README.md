📝 Django User Authentication & To-Do List App
This is a simple Django web application that includes user authentication (signup, login, logout) and a personal to-do list for each user. It demonstrates how to build secure and functional user flows with Django's built-in authentication system.

🚀 Features
✅ User Registration (Signup)

🔐 Secure Login & Logout

🗒️ Create, Read, Update, and Delete (CRUD) To-Do items

👤 Each user sees only their own tasks

🗓️ Deadline and completion tracking (optional)

🎨 Simple, responsive UI (HTML/CSS/Bootstrap)
